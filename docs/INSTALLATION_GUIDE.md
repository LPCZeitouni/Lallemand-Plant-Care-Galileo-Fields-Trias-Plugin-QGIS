
# Installation Guide

## ⚠️ Clean Installation Required

To ensure proper installation, start from a clean environment.

### Existing installations

- QGIS → MUST be uninstalled
- R / RStudio → strongly recommended

### Remove user configuration

Delete:

C:\Users\<YourUserName>\AppData\Roaming\QGIS\

(Optional for R):
C:\Users\<YourUserName>\Documents\R\

---

## Software Installation Order

1. R 4.3.1
2. RStudio 2024
3. RTOOLS43
4. QGIS 3.22.5

---

## QGIS Configuration

Install:

- Smart Map
- Processing R Provider

---

## Automatic Installer (Recommended)

### Step 1

Copy installer folder to:

C:\

### Step 2

Run:

InstaladorLallemand.exe

---

### Step 3 (User action)

1. Open QGIS  
2. Install **Processing R Provider**  
3. Close QGIS  

➡️ Installation continues automatically

---

## Final Check

- Open QGIS
- Plugin appears
- Launch plugin

# Guia do Usuário

## Objetivo

Este plugin processa dados de ensaios em faixas usando QGIS.

---

## Fluxo

1. Registro de dados
2. Carregar arquivos GPS
3. Criar polígonos
4. Filtrar pontos
5. Amostragem
6. Krigagem
7. RMSE
8. Correção de erro
9. Superfície de ganho
10. Mapas
11. Relatório

---

## Pontos importantes

- nome do arquivo deve conter "GPS"
- Smart Map necessário
- usar "Move layer to group"
- seguir ordem

---

## Resultado

- mapas
- análise
- relatório

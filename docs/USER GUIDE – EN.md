# User Guide

## Purpose

This plugin processes strip trial data using QGIS.

---

## Workflow

1. Data registration
2. Load GPS files
3. Create polygons
4. Filter points
5. Sampling
6. Kriging
7. RMSE
8. Error correction
9. Gain surface
10. Maps
11. Report

---

## Key Points

- GPS filename must contain "GPS"
- Smart Map required
- Use "Move layer to group"
- Follow workflow

---

## Output

- maps
- analysis
- report
- presentation

# Lallemand Geostat Field Trial Treatments

## Overview
Lallemand Geostat Field Trial Treatments is a QGIS plugin designed for advanced analysis of agronomic field trials.

It integrates geostatistical approaches such as kriging with R-based statistical processing to automate and accelerate trial analysis workflows.

## Features

- Import and manage field trial datasets
- Statistical processing using R
- Geostatistical modeling (kriging)
- Visualization of trial results
- Automation of repetitive analysis workflows

## Requirements

⚠️ Important:

This plugin currently requires a specific environment:

- QGIS 3.22
- R 4.3.2
- Required R and Python libraries (depending on configuration)

## Installation

### Method 1 – Local installation

1. Download the plugin ZIP file
2. Open QGIS
3. Go to *Plugins → Manage and Install Plugins*
4. Click *Install from ZIP*
5. Select the plugin archive

# Lallemand Plant Care Galileo Fields Trials Plugin for QGIS

## Overview

This QGIS plugin provides a complete workflow for processing agronomic strip trials in large fields, from raw data to final reports and presentations.

## Requirements

- QGIS 3.22.5
- R 4.3.1
- RStudio 2024
- RTOOLS43

Required QGIS plugins:

- Smart Map
- Processing R Provider

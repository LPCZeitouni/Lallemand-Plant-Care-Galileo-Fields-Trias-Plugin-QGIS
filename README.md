# Lallemand Plant Care – Galileo Field Trials Plugin (QGIS)

## Overview

This plugin provides a complete workflow for processing agronomic field trials based on geostatistical methods (Kriging).

It supports data preparation, spatial analysis, and automatic generation of maps and reports.

---

## Features

- GPS data processing
- Field polygon generation
- Sampling (80/20)
- Kriging interpolation
- RMSE analysis
- Error surface calculation
- Gain/loss maps
- Automated report generation
